# Engine English Documentation

This folder stores English, human-readable documentation for `@venus/engine`.

## Start Here

- `api/README.md`: categorized API catalog with entry demos.
- `api-reference.md`: public package API reference.
- `scene-object-model.md`: current render-facing scene object contract.

## Language Policy

- English docs live under `docs/en`.
- AI-facing rules live under `AI/`.
- Non-English human docs may stay under `docs/` until migrated into language-specific folders.
