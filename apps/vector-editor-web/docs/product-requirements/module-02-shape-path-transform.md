# 模块 02：形状、路径与变换

## 1. 模块目标

打造专业级图形编辑主链路：

- 形状创建
- 路径编辑
- 变换与吸附
- 对齐与分布

## 2. 用户故事

1. 作为视觉设计师，我希望拖拽创建形状时支持常见约束（等比、中心缩放）。
2. 作为插画师，我希望路径锚点编辑足够精细，支持快速修形。
3. 作为 UI 设计师，我希望移动对象时自动吸附关键边线与中心线。
4. 作为团队成员，我希望多对象对齐/分布结果稳定可复现。

## 3. 需求条目

### R1 形状创建一致性

1. 所有基础图形工具遵循统一创建语义：

- 点击创建默认尺寸
- 拖拽创建自定义尺寸
- 修饰键控制约束

2. 创建后立即进入可编辑状态。

验收标准：

- 7 类基础图形（矩形/椭圆/多边形/星形/线段/路径/文本）行为一致。

### R2 路径编辑增强

1. 支持锚点新增/删除。
2. 支持角点/平滑点类型转换。
3. 支持路径段分割与合并。
4. 支持路径闭合/断开。

验收标准：

- 在 200+ 点路径下编辑反馈稳定。
- 锚点/控制柄操作不引发几何跳变。

### R3 变换预览与提交分离

1. 移动、缩放、旋转采用预览态。
2. 释放指针时统一提交 patch。
3. 支持 ESC 取消当前变换会话。

验收标准：

- 高频拖拽期间无明显卡顿。
- 取消后文档与历史栈保持一致。

### R4 吸附与辅助线

1. 吸附来源至少包含：

- 对象边
- 对象中心
- 画布中心

2. 支持阈值配置与临时禁用。
3. 显示吸附辅助线与吸附点提示。

验收标准：

- 吸附触发可解释，且误吸附可控。
- 辅助线显示与实际吸附结果一致。

### R5 对齐与分布

1. 支持 6 向对齐（左/中/右/上/中/下）。
2. 支持横向/纵向分布。
3. 明确参考系（选区边界、首对象、画板）。

验收标准：

- 同一选择集重复执行结果一致。
- 复杂对象（含旋转）对齐规则可解释。

## 4. 对标提升项

1. 增加栅格与参考线系统（显示/吸附开关）。
2. 增加高级路径能力（复合路径、偏移路径）。
3. 增加可视化变换数值面板（精确输入）。

## 5. 指标

1. 变换流畅度

- 目标：典型文件中拖拽变换帧率稳定。

2. 吸附满意度

- 目标：用户反馈中的“吸附不准”问题下降 50%。
