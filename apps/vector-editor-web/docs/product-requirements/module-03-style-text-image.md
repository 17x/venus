# 模块 03：样式、文字与图片

## 1. 模块目标

提供专业、连贯、可扩展的视觉属性编辑能力，覆盖：

- 填充/描边/阴影
- 文字排版与文字编辑
- 图片裁剪与遮罩配合
- 属性面板一致性

## 2. 用户故事

1. 作为品牌设计师，我希望快速管理对象的填充、描边、透明度与效果。
2. 作为 UI 设计师，我希望文字编辑支持基础与进阶排版能力。
3. 作为内容设计师，我希望图片与蒙版组合工作流简单且可逆。

## 3. 需求条目

### R1 样式系统统一

1. 属性面板中样式编辑入口统一（Fill/Stroke/Effects）。
2. 样式修改采用实时预览 + 可撤销提交。
3. 支持多选对象的混合态展示。

验收标准：

- 多选时字段状态明确（统一值/混合值）。
- 样式变更可被历史完整回放。

### R2 填充与描边增强

1. 填充支持：纯色、线性渐变、径向渐变。
2. 描边支持：宽度、颜色、端点样式、箭头样式。
3. 支持复制/粘贴样式。

验收标准：

- 渐变编辑器可视化控制点交互可用。
- 描边渲染与属性值一致。

### R3 文字系统升级

1. 支持内容编辑与基础排版。
2. 支持文本 run 级样式覆盖。
3. 补齐段落能力：段前后距、对齐、缩进、行距策略。

验收标准：

- 中英文混排可稳定编辑。
- 文字样式切换不丢失 run 信息。

### R4 图片编辑能力

1. 支持图片导入、替换、基础变换。
2. 与蒙版能力打通（创建/清除/反选流程）。
3. 支持基础元信息展示（尺寸、比例、来源）。

验收标准：

- 图片替换不破坏现有布局与蒙版关系。
- 蒙版清除后结构可恢复。

## 4. 对标差距与规划

P0：

1. 补齐混合态样式编辑 UX。
2. 补齐文本内容编辑在属性面板与画布内的一致性。

P1：

1. 增加混合模式（Multiply/Screen 等）与不透明度系统。
2. 增加段落样式与文字样式集。

P2：

1. 文字沿路径。
2. 样式资产库（全局颜色、全局文本样式）。
