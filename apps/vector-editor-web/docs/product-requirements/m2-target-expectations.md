# M2 目标期望定义（可验收）

目标范围：

1. 路径编辑深度能力可用于复杂图形场景。
2. 样式/文字编辑在多选与混排场景下保持一致可解释。
3. 图层结构、蒙版、布尔链路可持续编辑且可回放。

## 1. 路径能力目标

1. 锚点编辑一致性

- 锚点新增/删除、角点/平滑点转换语义统一。
- 控制柄联动策略可预测，不产生突变。

2. 路径拓扑操作

- 分割、闭合、断开流程支持撤销/重做。
- 操作后拓扑与命中行为稳定。

验收标准：

1. 200+ 点路径操作稳定，无几何跳变。
2. 路径拓扑操作可回放且结果一致。

## 2. 样式与文字目标

1. 样式混合态

- 多选对象属性面板明确区分统一值与混合值。
- 样式提交可完整进入历史栈。

2. 文字 run 与段落

- run 样式覆盖与段落参数可同时存在。
- 中英文混排编辑与样式切换稳定。

验收标准：

1. 混合态字段反馈一致且可理解。
2. 文字样式切换不丢失 run 信息。

## 3. 结构与布尔目标

1. 图层与画布一致性

- 图层树选择、重排与画布选中状态一致。

2. 蒙版与布尔可编辑性

- 蒙版关系可视化可追踪。
- 布尔结果支持继续路径编辑。

验收标准：

1. 蒙版撤销/重做无结构损坏。
2. 布尔结果拓扑稳定且可继续编辑。

## 4. NFR 目标

1. 性能：路径与样式高频操作保持稳定反馈。
2. 可靠性：预览态/提交态与结构状态一致。
3. 可观测性：关键路径日志完整率 >= 95%。

验收标准：

1. M2 专项回归集持续通过。
2. 类型检查持续通过（`tsc -p tsconfig.app.json --noEmit`）。
