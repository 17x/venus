# 模块 04：图层、编组、蒙版与布尔

## 1. 模块目标

建立稳定的对象结构管理能力，确保复杂文档可控：

- 图层树
- 编组结构
- 蒙版关系
- 布尔结果可管理

## 2. 用户故事

1. 作为设计师，我希望在图层面板快速定位、选择和重排对象。
2. 作为插画师，我希望布尔运算后还能继续高效编辑。
3. 作为 UI 设计师，我希望蒙版关系可视化清晰，避免误操作。

## 3. 需求条目

### R1 图层树交互

1. 支持单选、多选、范围选择。
2. 支持拖拽重排与层级变更。
3. 支持锁定、隐藏、命名与快速筛选。

验收标准：

- 图层树操作与画布选择状态一致。
- 大量图层下滚动与选择保持流畅。

### R2 编组与解组

1. 编组保留相对位置与层级。
2. 解组后子节点恢复到预期父层级。
3. 支持嵌套组与隔离编辑。

验收标准：

- 深层嵌套组下操作不丢失子节点。
- 隔离编辑退出后状态一致。

### R3 蒙版模型

1. 支持创建蒙版、清除蒙版、替换蒙版源对象。
2. 蒙版关系可在图层树可视化。
3. 命中策略遵循蒙版可见区域。

验收标准：

- 蒙版前后对象可见性符合预期。
- 撤销/重做蒙版操作无结构损坏。

### R4 布尔运算

1. 支持并集、差集、交集。
2. 支持对多选对象一次性执行。
3. 结果对象支持继续路径编辑。

验收标准：

- 常见几何组合无异常碎片。
- 运算结果拓扑稳定并可再次布尔。

## 4. 对标差距

1. 布尔链路可视化编辑（非破坏式）尚未完整。
2. 图层搜索、标签、批处理能力不足。
3. 复杂蒙版场景的可视化反馈待增强。
