# `@venus/ui`

Package-scoped notes for shared editor UI primitives and chrome building
blocks.

## Stable Knowledge

- `@venus/ui` is the shared UI surface for active editor apps.
- Prefer reusable primitives and wrappers here over app-local duplicates when
  the behavior is not product-specific.

## Recent Updates

### 2026-04-08

- Shared shadcn/Radix-aligned primitives now back dialog, select, scroll area,
  tooltip, and editor chrome reuse in the vector app.
